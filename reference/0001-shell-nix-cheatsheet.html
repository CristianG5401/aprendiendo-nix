shell.nix básico

Plantilla mínima

let
  nixpkgs = fetchTarball "https://github.com/NixOS/nixpkgs/tarball/nixos-24.05";
  pkgs = import nixpkgs { config = {}; overlays = []; };
in

pkgs.mkShellNoCC {
  packages = with pkgs; [
    curl
    git
    jq
  ];

  MY_VAR = "valor";

  shellHook = ''
    echo "Shell lista"
  '';
}

Comandos

  • nix-shell: entra en la shell descrita por shell.nix.
  • exit: sale de la shell.
  • which jq: verifica desde dónde viene una herramienta.

Regla mental

Si lo necesitas para trabajar en ese proyecto o administrar ese entorno, decláralo en el archivo.
