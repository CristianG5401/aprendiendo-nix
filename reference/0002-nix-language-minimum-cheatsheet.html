Referencia rápida

Lenguaje Nix mínimo para leer shell.nix

Idea central

Nix evalúa expresiones para producir valores. No pienses primero en comandos secuenciales estilo Bash.

let ... in

let
  x = 1;
  y = 2;
in
  x + y

Analogía TS:

(() => {
  const x = 1;
  const y = 2;
  return x + y;
})()

Attribute set

{
  hostname = "homeserver";
  port = 22;
}

Como objeto JS, pero con = y ;.

Lista

[ "curl" "git" "jq" ]

Como array JS, pero sin comas.

Interpolación de strings

let name = "Nix"; in "Hola ${name}"

Similar a template literals en TS: `Hola ${name}`.

Acceso con punto

{ server = { hostname = "lab"; }; }.server.hostname

Similar a obj.server.hostname.

Comando de práctica

nix-instantiate --eval --strict archivo.nix

Evalúa un archivo .nix y muestra el valor resultante.
